# Metrics

Optimization metrics:

- initial metric
- best metric
- final accepted metric
- net gain
- gain per trial
- gain per accepted trial
- gain per hour
- gain per budget unit

Governance metrics:

- total trials
- kept, discarded, failed-invalid counts
- acceptance rate
- invalid rate
- editable-scope violation count
- complete provenance rate
- trials with rationale

Memory metrics:

- repeated bad count and rate
- repeated invalid count
- repeated degraded count
- raw memory size
- compressed context size
- compression ratio
