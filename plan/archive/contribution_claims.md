# Contribution Claims

## Claims

- Bounded execution improves auditability for autonomous experiment loops.
- Explicit lifecycle control makes autonomous experiments inspectable.
- Append-only memory and rationale can reduce repeated poor proposals under fixed budgets.
- Governed experimentation can produce measurable improvements on a real scientific ML node.

## Non-Claims

- This is not a general autonomous scientist.
- This is not proof of scientific discovery.
- This is not a universal optimization algorithm.
- This is not dependent on a specific coding-agent backend.
